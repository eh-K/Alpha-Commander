# Compiling Alpha Commander.
### 1. Install Visual Studio
>- Any version will work. It is up to you.

### 2. Download xdevkit.dll
>- It is required in order to compile.
>- Can be found [here.](https://github.com/eh-K/Alpha-Commander/blob/main/src/xdevkit.dll)

# Issues
### 1. Upon opening, Form11.Designer.cs appears. 
> In order to compile, delete this file before compiling.
